
Book Chapter Generator App - Requirements Document
----------------------------------------------------

**Overview:**
This application allows users to upload or paste book chapter text and generates:
- A concise summary
- Key takeaways
- Relevant tags
- A generated book chapter in a chosen format (PDF, DOCX, Markdown, Plain Text)

The application also allows for easy downloading and regenerating of the book chapter.

**Key Features:**
✅ Upload book chapter (PDF or text file) or paste text directly  
✅ Required fields: Chapter Title, Author Name, Book Name, Data Analyst's Notes, Additional Context  
✅ Generates summary, key takeaways, and tags  
✅ Select output format (PDF, DOCX, Markdown, Plain Text)  
✅ Download generated content in selected format  
✅ Regenerate the chapter output in a new format without re-inputting text  
✅ Clean and intuitive user interface with clear instructions  

**Technical Requirements:**
- Python 3.8 or above
- Libraries:
    • streamlit
    • PyPDF2 (for PDF text extraction)
    • google-generativeai (Gemini API SDK)

**External Dependencies:**

**Environment Variables:**
- None needed (API key is directly hardcoded for simplicity)

**How to Run:**
1. Install required Python packages:  
   pip install streamlit PyPDF2 google-generativeai

2. Run the Streamlit app:  
   streamlit run streamlit_app.py

**File Structure:**
- Single Python file (e.g., streamlit_app.py) containing all logic and UI.

**Data Analyst-Specific Fields in UI:**
- Chapter title  
- Author name  
- Book name  
- Data analyst's notes  
- Additional context (optional)

**Output:**
- Displays generated content directly in the app.  
- Allows download in PDF, DOCX, Markdown, or Plain Text format.

**Notes:**
- Uses Gemini API (gemini-2.0-flash) for content generation.  
- Easy-to-understand prompts provided to Gemini API for clear output.  
- App includes clear instructions and intuitive controls for a smooth user experience.
